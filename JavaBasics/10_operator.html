10. 연산자

연산자는 값을 계산, 비교, 대입, 판단할 때 사용하는 Java의 기본 문법입니다.
이 페이지에서는 산술 연산자, 대입 연산자, 증감 연산자, 비교 연산자,
논리 연산자, 삼항 연산자와 연산자 우선순위를 공부합니다.

이번 페이지의 학습 목표

연산자는 단순한 계산 기호가 아니라 프로그램의 값 흐름을 만드는 문법입니다.
값을 계산하고, 조건을 판단하고, 변수에 저장하고, 여러 조건을 연결하는 과정에서 연산자는 거의 모든 Java 코드에 사용됩니다.

연산자 개념 이해

연산자와 피연산자의 관계를 이해합니다.

계산과 대입

산술 연산자, 대입 연산자, 복합 대입 연산자를 익힙니다.

조건 판단

비교 연산자와 논리 연산자를 사용해 조건을 판단합니다.

실무 활용

가격 계산, 권한 검사, 페이지 계산, 평균 계산에 연산자를 적용합니다.

1. 핵심 개념 설명

1-1. 연산자와 피연산자

연산자는 값을 계산하거나 비교하거나 대입할 때 사용하는 기호입니다. 피연산자는 연산의 대상이 되는 값입니다. 

int result = 10 + 20;
10, 20 → 피연산자
+      → 연산자
result → 연산 결과를 저장하는 변수

1-2. 산술 연산자

산술 연산자는 숫자 계산에 사용합니다.

연산자 의미 예시 결과
+ 더하기 10 + 3 13
- 빼기 10 - 3 7
* 곱하기 10 * 3 30
/ 나누기 10 / 3 3
% 나머지 10 % 3 1 
int a = 10;
int b = 3;

System.out.println(a + b);
System.out.println(a - b);
System.out.println(a * b);
System.out.println(a / b);
System.out.println(a % b);

1-3. 나누기와 나머지 연산자

정수끼리 나누면 결과도 정수입니다. 소수점 아래는 버려집니다. 

System.out.println(10 / 3);
System.out.println(10 % 3);
3
1

소수점 결과를 얻고 싶다면 형 변환이 필요합니다. 

double result = (double) 10 / 3;
System.out.println(result);
3.3333333333333335

1-4. 문자열에서 + 연산자

+는 숫자끼리 사용하면 더하기지만, 문자열과 함께 사용하면 문자열 결합이 됩니다. 

System.out.println("점수: " + 90);
점수: 90

계산 순서에 주의해야 합니다.

System.out.println("결과: " + 10 + 20);
결과: 1020
System.out.println("결과: " + (10 + 20));
결과: 30

1-5. 대입 연산자

대입 연산자 =는 오른쪽 값을 왼쪽 변수에 저장합니다. 수학의 “같다”가 아니라 “저장한다”는 의미입니다. 

int age = 20;

=는 대입이고, ==는 비교입니다.

1-6. 복합 대입 연산자

복합 대입 연산자는 계산과 대입을 한 번에 수행합니다.

연산자 의미 같은 표현
+= 더해서 대입 a = a + b
-= 빼서 대입 a = a - b
*= 곱해서 대입 a = a * b
/= 나눠서 대입 a = a / b
%= 나머지를 대입 a = a % b 
int score = 80;

score += 10;

System.out.println(score);
90

1-7. 증감 연산자

증감 연산자는 값을 1 증가시키거나 1 감소시킵니다.

연산자 의미
++ 1 증가
-- 1 감소 
int count = 5;
count++;

System.out.println(count);
6

1-8. 전위 증감과 후위 증감

증감 연산자는 위치에 따라 값이 사용되는 시점이 달라집니다. 

++a
→ 먼저 1 증가
→ 증가한 값을 사용

a++
→ 먼저 현재 값을 사용
→ 나중에 1 증가

int a = 5;
int result = ++a;

System.out.println(a);
System.out.println(result);
6
6
int b = 5;
int result = b++;

System.out.println(b);
System.out.println(result);
6
5

1-9. 비교 연산자

비교 연산자는 두 값을 비교해서 true 또는 false를 반환합니다.

연산자 의미
== 같다
!= 같지 않다
> 크다
< 작다
>= 크거나 같다
<= 작거나 같다 
int age = 20;

System.out.println(age == 20);
System.out.println(age != 20);
System.out.println(age >= 19);

1-10. 논리 연산자

논리 연산자는 여러 조건을 연결하거나 조건을 반대로 바꿀 때 사용합니다.

연산자 의미 결과
&& AND, 그리고 양쪽이 모두 true일 때 true
|| OR, 또는 둘 중 하나라도 true면 true
! NOT, 부정 true를 false로, false를 true로 변경 
int age = 20;
boolean hasTicket = true;

if (age >= 19 && hasTicket) {
    System.out.println("입장 가능");
}

1-11. 단축 평가

&&와 ||는 결과가 이미 결정되면 뒤 조건을 평가하지 않습니다. 이것을 단축 평가라고 합니다. 

&&
앞 조건이 false이면 뒤 조건을 확인하지 않음

||
앞 조건이 true이면 뒤 조건을 확인하지 않음

if (user != null && user.isActive()) {
    System.out.println("활성 사용자");
}

user != null이 false이면 user.isActive()를 실행하지 않기 때문에 NullPointerException을 예방할 수 있습니다.

1-12. 삼항 연산자

삼항 연산자는 조건에 따라 둘 중 하나의 값을 선택합니다. 

조건식 ? 참일 때 값 : 거짓일 때 값
int age = 20;

String result = age >= 19 ? "성인" : "미성년자";

System.out.println(result);
성인

짧은 조건 대입에는 유용하지만, 조건이 복잡하면 if문을 사용하는 것이 더 읽기 좋습니다.

1-13. 연산자 우선순위

연산자는 계산 순서가 있습니다. 초급 단계에서는 우선순위를 모두 외우기보다 헷갈리면 괄호를 쓰는 습관이 가장 안전합니다. 

int result1 = 10 + 2 * 3;
int result2 = (10 + 2) * 3;
16
36

3. 산술 *, /, %
4. 산술 +, -
5. 비교 >, <, >=, <=
6. 동등 ==, !=

2. 그림처럼 이해하기

2-1. 산술 연산

10 + 3 = 13
10 - 3 = 7
10 * 3 = 30
10 / 3 = 3
10 % 3 = 1

2-2. 대입과 비교

age = 20
→ 20을 age에 저장

age == 20
→ age가 20과 같은지 비교
→ 결과는 true 또는 false

2-3. 논리 연산

&& AND
true && true   → true
true && false  → false

|| OR
true || false  → true
false || false → false

! NOT
!true  → false
!false → true

2-4. 삼항 연산자

age >= 19 ? "성인" : "미성년자"

조건이 true  → "성인"
조건이 false → "미성년자"

3. 코드 예제

3-1. 산술 연산자 예제

public class ArithmeticOperatorExample {

    public static void main(String[] args) {

        int a = 10;
        int b = 3;

        System.out.println(a + b);
        System.out.println(a - b);
        System.out.println(a * b);
        System.out.println(a / b);
        System.out.println(a % b);

    }

}

3-2. 복합 대입 연산자 예제

public class AssignmentOperatorExample {

    public static void main(String[] args) {

        int score = 80;

        score += 10;
        System.out.println(score);

        score -= 5;
        System.out.println(score);

        score *= 2;
        System.out.println(score);

    }

}

3-3. 증감 연산자 예제

public class IncrementOperatorExample {

    public static void main(String[] args) {

        int a = 5;

        System.out.println(++a);
        System.out.println(a);

        int b = 5;

        System.out.println(b++);
        System.out.println(b);

    }

}

3-4. 비교 연산자 예제

public class CompareOperatorExample {

    public static void main(String[] args) {

        int age = 20;

        System.out.println(age == 20);
        System.out.println(age != 20);
        System.out.println(age >= 19);
        System.out.println(age < 19);

    }

}

3-5. 논리 연산자 예제

public class LogicalOperatorExample {

    public static void main(String[] args) {

        int age = 20;
        boolean hasTicket = true;

        boolean canEnter = age >= 19 && hasTicket;

        System.out.println(canEnter);

        boolean isAdmin = false;
        boolean isOwner = true;

        boolean canEdit = isAdmin || isOwner;

        System.out.println(canEdit);

    }

}

3-6. 삼항 연산자 예제

public class TernaryOperatorExample {

    public static void main(String[] args) {

        int age = 20;

        String result = age >= 19 ? "성인" : "미성년자";

        System.out.println(result);

    }

}

4. 실무 연결

4-1. 가격 계산

int price = 10000;
int discount = 2000;

int finalPrice = price - discount;

4-2. 권한 조건 검사

boolean isLogin = true;
boolean hasPermission = false;

boolean canAccess = isLogin && hasPermission;

4-3. 페이지 계산

int currentPage = 1;
int nextPage = currentPage + 1;
int prevPage = currentPage - 1;

4-4. 짝수·홀수 판단

int number = 8;

boolean isEven = number % 2 == 0;

4-5. 추천 점수와 통계 계산

double averageScore = (double) totalScore / count;
boolean isRecommended = averageScore >= 80 && likeCount >= 5;

4-6. 실무형 조건 예제

public class PracticalOperatorExample {

    public static void main(String[] args) {

        int price = 10000;
        int discount = 2000;
        int finalPrice = price - discount;

        boolean isLogin = true;
        boolean hasPermission = true;

        boolean canBuy = isLogin && finalPrice > 0;
        boolean canAccessAdmin = isLogin && hasPermission;

        System.out.println("최종 가격: " + finalPrice);
        System.out.println("구매 가능 여부: " + canBuy);
        System.out.println("관리자 접근 가능 여부: " + canAccessAdmin);

    }

}

5. 자주 하는 실수

실수 1. =와 ==를 헷갈림

if (age = 20) {
}
if (age == 20) {
}

실수 2. 정수 나눗셈을 실수 결과로 착각

double result = 10 / 3;
double result = (double) 10 / 3;

실수 3. 문자열 결합과 숫자 덧셈을 헷갈림

System.out.println("결과: " + 10 + 20);
System.out.println("결과: " + (10 + 20));

실수 4. 증감 연산자를 복잡한 식에 섞어 씀

int a = 5;
int result = a++ + ++a;

읽기 쉬운 코드를 위해 증감 연산은 분리해서 사용하는 것이 좋습니다.

a++;
a++;
int result = a;

실수 5. 논리 조건에 괄호를 쓰지 않음

if (isLogin && isAdmin || isOwner) {
}
if ((isLogin && isAdmin) || isOwner) {
}

실수 6. &&와 &를 혼동함

if (user != null && user.isActive()) {
}

조건문에서는 보통 단축 평가가 가능한 &&, ||를 사용합니다.

6. 면접 포인트

Q1. 연산자와 피연산자는 무엇인가요?

답변: 연산자는 값을 계산하거나 비교하거나 대입할 때 사용하는 기호입니다.
피연산자는 연산의 대상이 되는 값입니다. 예를 들어 10 + 20에서 +는 연산자이고, 10과 20은 피연산자입니다.

Q2. /와 %의 차이는 무엇인가요?

답변: /는 나눗셈의 몫을 구하는 연산자이고, %는 나눗셈 후 나머지를 구하는 연산자입니다. 예를 들어 10 / 3은 3이고, 10 % 3은 1입니다.

Q3. =와 ==의 차이는 무엇인가요?

답변: =는 오른쪽 값을 왼쪽 변수에 저장하는 대입 연산자입니다. ==는 두 값이 같은지 비교하는 비교 연산자이며 결과는 true 또는 false입니다.

Q4. ++a와 a++의 차이는 무엇인가요?

답변: ++a는 값을 먼저 1 증가시킨 뒤 사용하고, a++는 현재 값을 먼저 사용한 뒤 1 증가시킵니다.
단독으로 사용할 때는 결과가 비슷하지만, 다른 연산식 안에서 사용하면 차이가 나타납니다.

Q5. 단축 평가란 무엇인가요?

답변: 단축 평가는 논리 연산에서 결과가 이미 결정되면 뒤 조건을 평가하지 않는 방식입니다.
&&는 앞 조건이 false이면 뒤 조건을 확인하지 않고, ||는 앞 조건이 true이면 뒤 조건을 확인하지 않습니다.

Q6. 삼항 연산자는 언제 사용하나요?

답변: 삼항 연산자는 조건에 따라 둘 중 하나의 값을 선택할 때 사용합니다. 간단한 조건 대입에는 유용하지만 조건이 복잡하면 if문을 사용하는 것이 더 읽기 좋습니다.

7. 오늘의 핵심 정리

  1. 연산자는 값을 계산, 비교, 대입, 판단하는 기호다.
  2. 산술 연산자는 +, -, *, /, %이다.
  3. 정수끼리 나누면 결과도 정수다.
  4. %는 나머지를 구하는 연산자다.
  5. =는 대입, ==는 비교다.
  6. += 같은 복합 대입 연산자는 계산과 대입을 동시에 한다.
  7. ++와 --는 값을 1 증가 또는 감소시킨다.
  8. 비교 연산자의 결과는 true 또는 false다.
  9. 논리 연산자는 &&, ||, !를 사용한다.
  10. &&와 ||는 단축 평가를 수행한다.
  11. 삼항 연산자는 조건에 따라 값을 선택한다.
  12. 연산자 우선순위가 헷갈리면 괄호를 사용한다.

학습 메모

이 페이지는 Java에서 값을 계산하고 조건을 판단하는 연산자 단원입니다.
산술 연산자, 대입 연산자, 비교 연산자, 논리 연산자, 삼항 연산자를 이해하면 이후 제어문, 반복문, 배열, 객체지향 로직을 훨씬 자연스럽게 작성할 수 있습니다.
특히 =와 ==, 정수 나눗셈, 문자열 결합, 단축 평가는 초급 단계에서 반드시 주의해야 합니다.
