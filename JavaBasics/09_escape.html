09. Escape 문자

Escape 문자는 문자열 안에서 줄바꿈, 탭, 따옴표, 역슬래시처럼
일반 문자로 표현하기 어려운 특수 문자를 나타내는 문법입니다.
이 페이지에서는 \n, \t, \", \', \\의 사용법을 공부합니다.

이번 페이지의 학습 목표

Escape 문자는 콘솔 출력, 문자열 작성, 파일 경로 표현, JSON 문자열 작성 등에서 자주 사용됩니다.
문자열 안에서 줄바꿈을 넣거나, 따옴표를 문자로 출력하거나, 역슬래시 자체를 표현하려면 Escape 문자의 역할을 정확히 이해해야 합니다.

Escape 문자 이해

역슬래시 \가 뒤의 문자를 특별하게 해석하게 만드는 구조를 이해합니다.

출력 제어

\n과 \t를 사용해 줄바꿈과 탭 간격을 표현합니다.

특수문자 출력

\", \', \\를 사용해 따옴표와 역슬래시를 출력합니다.

실무 활용

Windows 경로, 로그 메시지, JSON 문자열에서 Escape 문자가 쓰이는 상황을 익힙니다.

1. 핵심 개념 설명

1-1. Escape 문자란?

Escape 문자는 문자열 안에서 일반 문자로 쓰기 어려운 문자를 표현하기 위한 특수 문법입니다.
Java 문자열은 큰따옴표로 감싸기 때문에, 문자열 안에서 큰따옴표 자체를 출력하려면 특별한 처리가 필요합니다. 

System.out.println("그가 말했다. "안녕하세요"");

위 코드는 오류가 발생합니다. Java가 두 번째 큰따옴표에서 문자열이 끝났다고 판단하기 때문입니다. 

System.out.println("그가 말했다. \"안녕하세요\"");

출력 결과는 다음과 같습니다.

그가 말했다. "안녕하세요"

정리: Escape 문자는 문자열 안에서 특수한 의미를 가지는 문자를 안전하게 표현하기 위한 약속입니다.

1-2. Escape 문자의 핵심은 역슬래시

Escape 문자는 대부분 역슬래시 \로 시작합니다. 역슬래시는 뒤에 오는 문자를 일반 문자와 다르게 해석하라는 신호입니다. 

\ + 문자
↓
특별한 의미

\n  → 줄바꿈
\t  → 탭 간격
\"  → 큰따옴표
\'  → 작은따옴표
\\  → 역슬래시
Escape 문자 의미 예시
\n 줄바꿈 "Hello\nJava"
\t 탭 간격 "이름\t나이"
\" 큰따옴표 출력 "\"Java\""
\' 작은따옴표 출력 "\'A\'"
\\ 역슬래시 출력 "C:\\Java"

1-3. \n 줄바꿈

\n은 문자열 안에서 줄바꿈을 의미합니다. 한 문자열 안에서 원하는 위치에 줄을 바꾸고 싶을 때 사용합니다. 

System.out.println("첫 번째 줄\n두 번째 줄");

출력 결과는 다음과 같습니다.

첫 번째 줄
두 번째 줄
System.out.println("이름: 홍길동\n나이: 20\n직업: 학생");

출력 결과는 다음과 같습니다.

이름: 홍길동
나이: 20
직업: 학생

1-4. \t 탭 간격

\t는 탭 간격을 의미합니다. 콘솔에서 간단한 표 형태로 값을 보여줄 때 사용할 수 있습니다. 

System.out.println("이름\t나이\t점수");
System.out.println("홍길동\t20\t90");
System.out.println("김철수\t21\t85");

단, 콘솔 환경이나 글자 폭에 따라 완벽하게 정렬되지 않을 수 있습니다. 정확한 표 정렬이 필요하면 나중에 printf나 포맷팅 기능을 함께 사용합니다.

1-5. \" 큰따옴표 출력

문자열은 큰따옴표로 시작하고 끝납니다. 그래서 문자열 안에서 큰따옴표 자체를 출력하려면 \"를 사용해야 합니다. 

System.out.println("문자열은 \"Hello\"처럼 작성한다.");

출력 결과는 다음과 같습니다.

문자열은 "Hello"처럼 작성한다.

1-6. \' 작은따옴표 출력

문자열 안에서는 작은따옴표를 그냥 써도 되는 경우가 많습니다. 하지만 char에서 작은따옴표 자체를 문자로 저장하려면 Escape 처리가 필요합니다. 

System.out.println("문자형 char는 'A'처럼 작성한다.");
char quote = '\'';
System.out.println(quote);

출력 결과는 다음과 같습니다.

'

1-7. \\ 역슬래시 출력

Java에서 역슬래시 \는 Escape 문자의 시작 기호입니다. 그래서 역슬래시 자체를 출력하려면 \\처럼 두 번 작성해야 합니다. 

System.out.println("C:\\Java\\workspace");

출력 결과는 다음과 같습니다.

C:\Java\workspace

핵심: 코드에서는 \\라고 쓰지만, 실제 출력은 \ 하나입니다.

2. print, println, char와 Escape 문자

2-1. println()과 \n의 차이

println()은 출력이 끝난 뒤 줄바꿈을 수행하는 메서드입니다. 반면 \n은 문자열 내부에서 원하는 위치에 줄바꿈을 넣는 Escape 문자입니다. 

System.out.println("Hello");
System.out.println("Java");

출력 결과는 다음과 같습니다.

Hello
Java
System.out.print("Hello\n");
System.out.print("Java");

출력 결과는 다음과 같습니다.

Hello
Java
println()
→ 출력이 끝난 뒤 줄바꿈

\n
→ 문자열 중간 원하는 위치에서 줄바꿈

2-2. Escape 문자와 char

Escape 문자는 문자열뿐 아니라 char에서도 사용할 수 있습니다. char는 문자 하나만 저장할 수 있지만, '\n' 같은 Escape 문자는 하나의 특수 문자로 취급됩니다. 

char newLine = '\n';
char tab = '\t';
char singleQuote = '\'';
char backslash = '\\';
char ch = 'A';    // 가능
char ch2 = '\n';  // 가능
char ch3 = 'AB';  // 오류

2-3. 문자열 결합과 Escape 문자

Escape 문자는 문자열 결합과 함께 사용할 수 있습니다. 변수 값과 줄바꿈 또는 탭을 함께 조합해 출력할 수 있습니다. 

String name = "홍길동";
int age = 20;

System.out.println("이름: " + name + "\n나이: " + age);

출력 결과는 다음과 같습니다.

이름: 홍길동
나이: 20
System.out.println("이름\t나이\t점수");
System.out.println(name + "\t" + age + "\t" + 90);

4-5. char에서 Escape 문자 사용

public class CharEscapeExample {

    public static void main(String[] args) {

        char newLine = '\n';

        char singleQuote = '\'';
        char backslash = '\\';

        System.out.println(backslash);

    }

}

5. 실무 연결

5-1. Windows 파일 경로

Windows 경로는 역슬래시를 사용합니다. 하지만 Java 문자열에서 역슬래시는 Escape 시작 기호이므로, 실제 역슬래시를 표현하려면 두 번 작성해야 합니다. 

String path = "C:\Users\ICT\Desktop";
String path = "C:\\Users\\ICT\\Desktop";

또는 슬래시 /를 사용할 수도 있습니다.

String path = "C:/Users/ICT/Desktop";

5-2. 로그 메시지 출력

로그나 디버깅 출력에서 여러 줄 정보를 보기 좋게 표시할 때 \n을 사용할 수 있습니다. 

System.out.println("요청 정보\n사용자 ID: " + userId + "\n요청 경로: " + path);

출력 예시는 다음과 같습니다.

요청 정보
사용자 ID: 10
요청 경로: /api/users

5-3. JSON 문자열 작성

JSON 문자열 안에는 큰따옴표가 많이 들어갑니다. 따라서 Java 문자열 안에서 JSON을 직접 작성할 때는 \"가 자주 등장합니다. 

String json = "{\"name\":\"홍길동\", \"age\":20}";
System.out.println(json);

다만 실무에서는 JSON 문자열을 직접 조립하기보다 Jackson 같은 JSON 라이브러리를 사용하는 경우가 많습니다.

5-4. 콘솔 메뉴와 안내문 구성

콘솔 프로그램에서는 메뉴나 안내문을 여러 줄로 보여주기 위해 Escape 문자를 사용할 수 있습니다. 

System.out.println("메뉴\n1. 회원가입\n2. 로그인\n3. 종료");

출력 결과는 다음과 같습니다.

메뉴
1. 회원가입
2. 로그인
3. 종료

5-5. Text Block과의 연결

최신 Java에서는 여러 줄 문자열을 더 편하게 작성할 수 있는 Text Block 기능도 있습니다. 하지만 기본 문법 학습 단계에서는 먼저 Escape 문자를 정확히 이해하는 것이 중요합니다. 

String message = """
        이름: 홍길동
        나이: 20
        직업: 학생
        """;

학습 순서: 먼저 \n, \t, \", \\를 익힌 뒤, 여러 줄 문자열이 많아질 때 Text Block을 함께 이해하면 좋습니다.

6. 자주 하는 실수

실수 1. 문자열 안 큰따옴표를 Escape 처리하지 않음

System.out.println("그가 말했다. "안녕"");

올바른 예시는 다음과 같습니다.

System.out.println("그가 말했다. \"안녕\"");

실수 2. Windows 경로에서 역슬래시를 하나만 씀

String path = "C:\Java\workspace";

올바른 예시는 다음과 같습니다.

String path = "C:\\Java\\workspace";

또는 아래처럼 쓸 수 있습니다.

String path = "C:/Java/workspace";

실수 3. \n 자체를 출력하고 싶은데 줄바꿈이 됨

System.out.println("줄바꿈 기호는 \n 입니다.");

문자 그대로 \n을 출력하려면 역슬래시도 Escape 처리해야 합니다. 

System.out.println("줄바꿈 기호는 \\n 입니다.");

출력 결과는 다음과 같습니다.

줄바꿈 기호는 \n 입니다.

실수 4. char에 여러 글자를 넣음

char ch = 'AB';

올바른 예시는 다음과 같습니다.

char ch = 'A';
            String str = "AB";

실수 5. 문자열 결합과 계산 순서를 헷갈림

System.out.println("합계:\t" + 10 + 20);

출력 결과는 다음과 같습니다.

합계:    1020

계산을 먼저 하려면 괄호를 사용합니다.

System.out.println("합계:\t" + (10 + 20));

출력 결과는 다음과 같습니다.

합계:    30

7. 면접 포인트

Q1. Escape 문자란 무엇인가요?

답변: Escape 문자는 문자열 안에서 줄바꿈, 탭, 따옴표, 역슬래시처럼 일반적으로 표현하기 어려운 문자를 나타내기 위한 특수 문자입니다. Java에서는 보통 역슬래시 \로 시작합니다.

Q2. \n과 println()의 차이는 무엇인가요?

답변: println()은 출력이 끝난 뒤 줄바꿈을 수행하는 메서드이고, \n은 문자열 내부에서 원하는 위치에 줄바꿈을 넣는 Escape 문자입니다.

Q3. 문자열 안에서 큰따옴표를 출력하려면 어떻게 하나요?

답변: 큰따옴표는 문자열의 시작과 끝을 나타내기 때문에, 문자열 내부에서 출력하려면 \"처럼 Escape 처리해야 합니다.

Q4. Java 문자열에서 Windows 경로를 작성할 때 주의할 점은 무엇인가요?

답변: 역슬래시 \는 Escape 문자의 시작 기호이므로, Windows 경로를 Java 문자열로 작성할 때는 \\처럼 두 번 써야 합니다. 또는 슬래시 /를 사용할 수도 있습니다.

Q5. \\n과 \n의 차이는 무엇인가요?

답변: \n은 실제 줄바꿈을 의미합니다. \\n은 역슬래시와 n 문자를 그대로 출력하기 위한 표현입니다.

Q6. Escape 문자는 문자열에서만 사용할 수 있나요?

답변: 아닙니다. Escape 문자는 char에서도 사용할 수 있습니다. 예를 들어 '\n', '\t', '\'', '\\'처럼 문자 하나를 표현할 때도 사용할 수 있습니다.

8. 오늘의 핵심 정리

  1. Escape 문자는 문자열 안에서 특수한 문자를 표현하기 위한 문법이다.
  2. Escape 문자는 보통 역슬래시 \로 시작한다.
  3. \n은 줄바꿈이다.
  4. \t는 탭 간격이다.
  5. \"는 큰따옴표 출력이다.
  6. \'는 작은따옴표 출력이다.
  7. \\는 역슬래시 출력이다.
  8. Windows 경로에서는 역슬래시를 \\처럼 두 번 써야 한다.
  9. 문자열 안에서 \n 자체를 출력하려면 \\n처럼 작성한다.
  10. Escape 문자는 문자열뿐 아니라 char에서도 사용할 수 있다.

학습 메모

이 페이지는 Java 문자열 안에서 특수한 표현을 다루는 Escape 문자 단원입니다.
\n, \t, \", \', \\를 정확히 이해하면 콘솔 출력, 파일 경로, 로그 메시지, JSON 문자열, 안내문 작성에서 발생하는 오류를 줄일 수 있습니다.
특히 역슬래시는 Escape 문자의 시작 기호이므로, 역슬래시 자체를 출력할 때는 반드시 \\처럼 작성해야 합니다.
