11. 제어문

제어문은 프로그램의 실행 흐름을 조절하는 문법입니다.
이 페이지에서는 조건문인 if, switch, 반복문인 for, while, do-while,
그리고 흐름을 중간에 바꾸는 break, continue를 공부합니다.

이번 페이지의 학습 목표

Java 프로그램은 기본적으로 위에서 아래로 순서대로 실행됩니다.
하지만 실제 프로그램에서는 조건에 따라 다른 코드를 실행하거나, 같은 코드를 여러 번 반복하거나, 반복 도중 특정 시점에서 멈추거나 건너뛰어야 합니다.
이때 사용하는 문법이 제어문입니다.

조건문 이해

if, if-else, else if, switch의 흐름을 이해합니다.

반복문 이해

for, while, do-while의 차이를 익힙니다.

흐름 제어

break와 continue가 반복 흐름을 어떻게 바꾸는지 이해합니다.

실무 적용

로그인 처리, 권한 분기, 목록 출력, 검색 종료, 데이터 필터링에 제어문을 적용합니다.

1. 제어문의 큰 구조

1-1. 제어문이란?

제어문은 프로그램의 실행 흐름을 제어하는 문법입니다. Java 코드는 기본적으로 위에서 아래로 실행되지만, 제어문을 사용하면 조건에 따라 실행 흐름을 나누거나, 특정 코드를 반복해서 실행할 수 있습니다. 

System.out.println("1번");
System.out.println("2번");
System.out.println("3번");

위 코드는 순서대로 실행됩니다. 그러나 실제 프로그램은 다음과 같은 판단이 필요합니다.

  • 로그인 성공이면 메인 화면으로 이동한다.
  • 로그인 실패이면 오류 메시지를 출력한다.
  • 점수가 60점 이상이면 합격으로 처리한다.
  • 게시글 목록 개수만큼 반복해서 화면에 출력한다.
  • 검색 중 원하는 데이터를 찾으면 반복을 중단한다.

1-2. 교안 기준 제어문의 분류

교안에서는 제어문을 조건문/분기문, 반복문, 기타 흐름 제어문으로 나누고 있습니다. 

제어문
├─ 조건문 / 분기문
│  ├─ if문
│  └─ switch문
│
├─ 반복문
│  ├─ for문       : 구간 반복
│  ├─ while문     : 조건 반복
│  └─ do-while문  : 실행 반복
│
└─ 기타 흐름 제어문
├─ continue문
└─ break문

핵심: 조건문은 실행할 코드를 선택하고, 반복문은 같은 코드를 여러 번 실행하며, break와 continue는 반복 흐름을 중간에 바꿉니다.

2. 조건문

2-1. if문

if문은 조건식이 true일 때만 블록 안의 코드를 실행합니다. 조건식은 반드시 결과가 true 또는 false가 되는 식이어야 합니다. 

if (조건식) {
    조건식이 true일 때 실행할 코드
}
int age = 20;

if (age >= 19) {
    System.out.println("성인입니다.");
}
조건 검사
↓
true  → if 블록 실행
false → if 블록 건너뜀

2-2. 조건식이란?

조건식은 결과가 boolean인 식입니다. 비교 연산자와 논리 연산자를 사용해 조건을 만들 수 있습니다. 

age >= 19
score >= 60
isLogin == true
userType.equals("ADMIN")
isLogin && isAdmin

Java에서는 숫자 자체를 조건식으로 사용할 수 없습니다.

if (age) {
}
if (age >= 19) {
}

2-3. if-else문

if-else문은 조건이 참일 때와 거짓일 때 실행 흐름을 나눕니다. 

int age = 17;

if (age >= 19) {
    System.out.println("성인입니다.");
} else {
    System.out.println("미성년자입니다.");
}
조건 검사
↓
true  → if 블록 실행
false → else 블록 실행

2-4. else if문

조건이 여러 개일 때는 else if를 사용합니다. 위에서부터 차례대로 검사하고, 처음 true가 된 블록만 실행합니다. 

int score = 85;

if (score >= 90) {
    System.out.println("A");
} else if (score >= 80) {
    System.out.println("B");
} else if (score >= 70) {
    System.out.println("C");
} else {
    System.out.println("F");
}

출력 결과는 다음과 같습니다.

B
조건1 검사
├─ true  → 조건1 블록 실행 후 종료
└─ false → 조건2 검사
            ├─ true  → 조건2 블록 실행 후 종료
            └─ false → 다음 조건 또는 else 검사

2-5. else if 조건 순서

else if는 위에서부터 순서대로 검사하기 때문에 조건의 순서가 중요합니다. 범위가 겹치는 조건은 더 구체적인 조건을 위에 두어야 합니다. 

int score = 95;

if (score >= 60) {
    System.out.println("합격");
} else if (score >= 90) {
    System.out.println("우수");
}

위 코드는 score >= 60에서 이미 true가 되므로 우수가 출력되지 않습니다. 

if (score >= 90) {
    System.out.println("우수");
} else if (score >= 60) {
    System.out.println("합격");
}

2-6. switch문

switch문은 하나의 값에 따라 여러 경우 중 하나를 선택할 때 사용합니다. 메뉴 번호, 상태 코드, 권한 코드처럼 값이 명확히 정해져 있는 경우에 적합합니다. 

int menu = 2;

switch (menu) {
    case 1:
        System.out.println("회원가입");
        break;
    case 2:
        System.out.println("로그인");
        break;
    case 3:
        System.out.println("종료");
        break;
    default:
        System.out.println("잘못된 메뉴입니다.");
}

출력 결과는 다음과 같습니다.

로그인
switch 값 확인
↓
case 1 일치? → 실행 후 break
case 2 일치? → 실행 후 break
case 3 일치? → 실행 후 break
일치 없음    → default 실행

2-7. switch문의 break와 default

switch문에서 break를 생략하면 아래 case까지 이어서 실행될 수 있습니다. 이런 현상을 fall-through라고 합니다. 

int menu = 1;

switch (menu) {
    case 1:
        System.out.println("회원가입");
    case 2:
        System.out.println("로그인");
    case 3:
        System.out.println("종료");
}

출력 결과는 다음처럼 의도와 다를 수 있습니다.

회원가입
로그인
종료

default는 어떤 case에도 해당하지 않을 때 실행되며, if문의 else와 비슷한 역할을 합니다.

2-8. if문과 switch문 비교

구분 if문 switch문
판단 방식 조건식 하나의 값
적합한 조건 범위, 비교, 논리 조합 특정 값 비교
예시 age >= 19 && hasTicket menu == 1
주 사용처 복잡한 조건 판단 메뉴, 상태값, 권한 코드

3. 반복문

3-1. 반복문이란?

반복문은 같은 코드를 여러 번 실행할 때 사용합니다. 같은 출력문을 여러 번 작성하는 대신 반복문으로 간결하게 표현할 수 있습니다. 

System.out.println("Java");
System.out.println("Java");
System.out.println("Java");
for (int i = 1; i <= 3; i++) {
    System.out.println("Java");
}

3-2. for문 — 구간 반복

교안에서는 for문을 구간 반복으로 설명합니다. 반복 횟수나 시작과 끝 범위가 명확할 때 주로 사용합니다. 

for (초기식; 조건식; 증감식) {
    반복 실행할 코드
}
for (int i = 1; i <= 5; i++) {
    System.out.println(i);
}

출력 결과는 다음과 같습니다.

1
2
3
4
5
초기식
↓
조건식 검사
↓ true
본문 실행
↓
증감식
↓
조건식 재검사
↓ false
반복 종료

3-3. for문의 구성 요소

요소 예시 의미
초기식 int i = 1 반복 변수를 처음 설정합니다.
조건식 i <= 5 true이면 반복하고 false이면 종료합니다.
증감식 i++ 반복이 한 번 끝난 뒤 반복 변수 값을 바꿉니다.

3-4. for문 예제 — 합계 구하기

int sum = 0;

for (int i = 1; i <= 10; i++) {
    sum += i;
}

System.out.println(sum);

출력 결과는 다음과 같습니다.

55
sum = 0
i = 1  → sum = 1
i = 2  → sum = 3
i = 3  → sum = 6
...
i = 10 → sum = 55

3-5. while문 — 조건 반복

교안에서는 while문을 조건 반복으로 설명합니다. 조건이 true인 동안 반복하며, 반복 횟수보다 조건 자체가 중요할 때 사용합니다. 

int count = 1;

while (count <= 5) {
    System.out.println(count);
    count++;
}

출력 결과는 다음과 같습니다.

1
2
3
4
5

3-6. while문의 무한 반복 주의

while문에서는 조건을 false로 만들 수 있는 변화가 필요합니다. 변화가 없으면 무한 반복이 발생할 수 있습니다. 

int count = 1;

while (count <= 5) {
    System.out.println(count);
}
int count = 1;

while (count <= 5) {
    System.out.println(count);
    count++;
}

3-7. do-while문 — 실행 반복

교안에서는 do-while문을 실행 반복으로 설명합니다. 먼저 한 번 실행하고 나중에 조건을 검사하기 때문에 최소 한 번은 실행됩니다. 

int count = 10;

do {
    System.out.println(count);
    count++;
} while (count <= 5);

출력 결과는 다음과 같습니다.

10
while
조건 검사 → true면 실행

do-while
먼저 실행 → 조건 검사

while은 0번 실행될 수 있음
do-while은 최소 1번 실행됨

3-8. while과 do-while 비교

구분 while do-while
조건 검사 먼저 검사 나중에 검사
최소 실행 횟수 0번 가능 최소 1번
사용 상황 조건이 맞을 때만 실행 일단 한 번 실행 후 조건 확인

4. break와 continue

4-1. break문

break는 반복문이나 switch문을 즉시 종료합니다. 반복 중 원하는 값을 찾았거나, 더 이상 반복할 필요가 없을 때 사용합니다. 

for (int i = 1; i <= 10; i++) {
    if (i == 5) {
        break;
    }

    System.out.println(i);
}

출력 결과는 다음과 같습니다.

1
2
3
4

4-2. continue문

continue는 반복문 전체를 종료하지 않고, 현재 반복만 건너뛰고 다음 반복으로 넘어갑니다. 

for (int i = 1; i <= 5; i++) {
    if (i == 3) {
        continue;
    }

    System.out.println(i);
}

출력 결과는 다음과 같습니다.

1
2
4
5

4-3. break와 continue 비교

구분 break continue
의미 반복문 전체 종료 현재 반복만 건너뜀
이후 흐름 반복문 밖으로 이동 다음 반복으로 이동
사용 예 검색 성공 시 종료 특정 값만 제외 
break
→ 반복문 전체 종료

continue

5. 중첩 반복문

5-1. 중첩 반복문이란?

반복문 안에 또 다른 반복문을 넣을 수 있습니다. 표, 좌표, 구구단, 2차원 배열을 다룰 때 자주 사용합니다. 

for (int i = 1; i <= 3; i++) {
    for (int j = 1; j <= 2; j++) {
        System.out.println("i=" + i + ", j=" + j);
    }
}

출력 결과는 다음과 같습니다.

i=1, j=1
i=1, j=2
i=2, j=1
i=2, j=2
i=3, j=1
i=3, j=2

5-2. 구구단 예제

for (int dan = 2; dan <= 9; dan++) {
    System.out.println("[" + dan + "단]");

    for (int num = 1; num <= 9; num++) {
        System.out.println(dan + " x " + num + " = " + (dan * num));
    }
}
바깥 for → 2단부터 9단까지
안쪽 for → 1부터 9까지 곱하기

6. 실무 연결

6-1. 로그인 성공/실패 처리

if (isLoginSuccess) {
    System.out.println("메인 페이지로 이동합니다.");
} else {
    System.out.println("아이디 또는 비밀번호가 일치하지 않습니다.");
}

6-2. 사용자 권한별 화면 분기

switch (userType) {
    case "ADMIN":
        System.out.println("관리자 페이지");
        break;
    case "PUBLISHER":
        System.out.println("공인중개사 페이지");
        break;
    case "USER":
        System.out.println("일반 회원 페이지");
        break;
    default:
        System.out.println("접근 권한을 확인할 수 없습니다.");
}

6-3. 목록 데이터 반복 출력

for (int i = 0; i < propertyList.size(); i++) {
    System.out.println(propertyList.get(i));
}

6-4. 검색 성공 시 반복 종료

for (int i = 0; i < users.length; i++) {
    if (users[i].equals("admin")) {
        System.out.println("관리자 계정을 찾았습니다.");
        break;
    }
}

6-5. 특정 데이터만 건너뛰기

for (int i = 0; i < scores.length; i++) {
    if (scores[i] < 0) {
        continue;
    }

    System.out.println(scores[i]);
}

6-6. 읽기 쉬운 제어문 만들기

제어문은 중첩이 깊어질수록 읽기 어려워집니다. 논리 연산자를 사용하거나 조건을 변수로 분리하면 더 읽기 쉬운 코드가 됩니다. 

if (isLogin) {
    if (isAdmin) {
        if (active) {
            System.out.println("접근 가능");
        }
    }
}
boolean canAccess = isLogin && isAdmin && active;

if (canAccess) {
    System.out.println("접근 가능");
}

7. 자주 하는 실수

실수 1. if 조건식 뒤에 세미콜론을 붙임

if (age >= 19);
{
    System.out.println("성인입니다.");
}
if (age >= 19) {
    System.out.println("성인입니다.");
}

실수 2. switch문에서 break를 빼먹음

switch (menu) {
    case 1:
        System.out.println("회원가입");
    case 2:
        System.out.println("로그인");
}
switch (menu) {
    case 1:
        System.out.println("회원가입");
        break;
    case 2:
        System.out.println("로그인");
        break;
}

실수 3. while문에서 증감식을 빼먹음

int i = 1;

while (i <= 5) {
    System.out.println(i);
}
int i = 1;

while (i <= 5) {
    System.out.println(i);
    i++;
}

실수 4. for문 조건을 반대로 작성함

for (int i = 1; i >= 5; i++) {
    System.out.println(i);
}
for (int i = 1; i <= 5; i++) {
    System.out.println(i);
}

실수 5. break와 continue를 헷갈림

if (i == 3) {
    break;
}

위 코드는 반복문 전체 종료입니다.

if (i == 3) {
    continue;
}

위 코드는 현재 반복만 건너뜁니다.

실수 6. else if 조건 순서를 잘못 배치함

int score = 95;

if (score >= 60) {
    System.out.println("합격");
} else if (score >= 90) {
    System.out.println("우수");
}
if (score >= 90) {
    System.out.println("우수");
} else if (score >= 60) {
    System.out.println("합격");
}

8. 면접 포인트

Q1. 제어문이란 무엇인가요?

답변: 제어문은 프로그램의 실행 흐름을 제어하는 문법입니다.
조건에 따라 다른 코드를 실행하거나, 특정 코드를 반복하거나, 반복 흐름을 중간에 종료 또는 건너뛰기 위해 사용합니다.

Q2. if문과 switch문의 차이는 무엇인가요?

답변: if문은 범위 비교, 논리 조합, 복잡한 조건에 적합합니다.
switch문은 하나의 값이 여러 case 중 어떤 값과 일치하는지 비교할 때 적합합니다. 예를 들어 나이 범위 판단은 if문이 좋고, 메뉴 번호나 상태 코드 분기는 switch문이 좋습니다.

Q3. for문과 while문의 차이는 무엇인가요?

답변: for문은 반복 횟수나 구간이 정해져 있을 때 주로 사용합니다. while문은 조건이 true인 동안 반복하므로 반복 횟수가 명확하지 않고 조건 중심일 때 사용합니다.

Q4. while문과 do-while문의 차이는 무엇인가요?

답변: while문은 조건을 먼저 검사한 뒤 실행하므로 한 번도 실행되지 않을 수 있습니다.
do-while문은 본문을 먼저 실행한 뒤 조건을 검사하므로 조건이 false여도 최소 한 번은 실행됩니다.

Q5. break와 continue의 차이는 무엇인가요?

답변: break는 반복문 전체를 종료합니다. continue는 현재 반복만 건너뛰고 다음 반복으로 이동합니다.

Q6. else if문에서 조건 순서가 중요한 이유는 무엇인가요?

답변: else if문은 위에서부터 조건을 검사하고, 처음 true가 된 블록만 실행합니다. 따라서 범위가 겹치는 조건은 더 구체적인 조건을 먼저 배치해야 합니다.

9. 오늘의 핵심 정리

  1. 제어문은 프로그램의 실행 흐름을 제어한다.
  2. 조건문에는 if문과 switch문이 있다.
  3. 반복문에는 for문, while문, do-while문이 있다.
  4. 기타 흐름 제어문에는 break문과 continue문이 있다.
  5. if문은 조건식이 true일 때 실행된다.
  6. switch문은 하나의 값에 따라 여러 case 중 하나를 선택한다.
  7. for문은 반복 횟수나 구간이 정해져 있을 때 적합하다.
  8. while문은 조건이 true인 동안 반복한다.
  9. do-while문은 최소 한 번 실행한 뒤 조건을 검사한다.
  10. break는 반복문이나 switch문을 종료한다.
  11. continue는 현재 반복만 건너뛰고 다음 반복으로 넘어간다.
  12. 중첩이 깊은 제어문은 조건을 정리해서 읽기 쉽게 만드는 것이 좋다.

이 페이지는 Java 프로그램의 실행 흐름을 제어하는 제어문 단원입니다.

조건문에는 if, switch, 반복문에는 for, while, do-while, 기타 흐름 제어문에는 continue, break가 포함됩니다.
