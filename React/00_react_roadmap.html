React.js With Vite · 00

React 학습 로드맵

React를 처음 실행하는 단계에서 시작해 Vite 프로젝트 구조를 이해하고,
JSX, 컴포넌트, props, state, Hook, Router, REST API, 배포까지 이어지는 프론트엔드 학습 흐름을 정리하는 첫 단원입니다.

1. React를 공부한다는 것

React는 화면을 작은 컴포넌트로 나누고, 데이터가 바뀌면 필요한 화면을 다시 그리는 방식으로 사용자 인터페이스를 만드는 JavaScript 라이브러리입니다.
이 교안에서는 단순히 React 문법만 외우는 것이 아니라, 프로젝트를 만들고, 파일 구조를 읽고, 화면을 컴포넌트로 분리하고, 상태 변화에 따라 화면이 바뀌는 흐름을 단계적으로 공부합니다.
최종적으로는 Router로 여러 화면을 연결하고, REST API 서버와 통신한 뒤, Vite 빌드 결과물을 배포하는 흐름까지 이해하는 것을 목표로 합니다.

2. 전체 학습 흐름

React 학습은 한 번에 모든 개념을 외우기보다, 아래 흐름처럼 프로젝트 실행부터 실제 앱 구조까지 넓혀 가는 방식이 좋습니다.

  1. 개발 환경 준비 Node.js, NPM, VS Code, React Developer Tools, Vite를 준비합니다.
  2. Vite 프로젝트 구조 이해 index.html, main.jsx, App.jsx, public, src/assets, package.json의 역할을 읽습니다.
  3. 화면 작성 방식 익히기 CSS 적용 방식, Bootstrap, JSX 문법을 통해 React 화면을 구성합니다.
  4. 렌더링 원리 이해 DOM, Reflow, Repaint, Virtual DOM, Diffing, Reconciliation의 흐름을 이해합니다.
  5. 컴포넌트와 데이터 흐름 컴포넌트, props, children, state를 통해 부모와 자식의 데이터 전달 구조를 익힙니다.
  6. Hook으로 동적인 화면 만들기 useState, useEffect, useRef, useReducer, useContext 등으로 상태와 생명주기를 관리합니다.
  7. 라우팅과 API 연동 React Router로 화면을 나누고, axios와 fetch로 REST API 서버와 통신합니다.
  8. 빌드와 배포 npm run build 결과물을 이해하고, Spring 프로젝트에 배포하는 흐름을 정리합니다.

5. 학습 후 할 수 있게 되는 것

  • Vite로 React 프로젝트를 생성하고 실행할 수 있습니다.
