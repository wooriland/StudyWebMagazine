01. HTML5 Document Structure

이 페이지에서는 웹 문서의 뼈대를 이루는 HTML5의 기본 구조를 공부합니다.
DOCTYPE, html, head, body, header, main, section, article 태그를 중심으로
의미 있는 웹 문서를 만드는 방법을 정리합니다.

HTML5란?

HTML5는 웹 문서의 구조를 더 의미 있게 표현하기 위한 웹 표준입니다.
기존보다 간결한 문서 선언과 시맨틱 태그를 제공하며,
멀티미디어와 다양한 입력 폼을 지원합니다.


시맨틱 태그 설명

시맨틱 태그는 태그 이름만 보고도 해당 영역의 의미를 알 수 있는 태그입니다.
예를 들어 header, nav, section, article, aside, footer가 있습니다.


section과 div의 차이

section

section은 제목과 본문을 가진 하나의 주제 영역을 나타낼 때 사용합니다.

div

div는 의미보다는 스타일 적용이나 레이아웃 구성을 위해 사용하는 일반적인 묶음 태그입니다.
