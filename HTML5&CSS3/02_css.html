02. CSS3 Style Sheet

이 페이지에서는 HTML 문서에 디자인을 입히는 CSS3의 기본 구조를 공부합니다.
selector, property, value, box model, layout 개념을 익히고,
학습 매거진 화면을 카드형 UI와 반응형 레이아웃으로 꾸미는 방법을 정리합니다.

CSS3란?

CSS는 HTML 문서에 디자인을 입히는 스타일 언어입니다. 글꼴, 색상, 여백, 배경, 테두리, 배치, 애니메이션 등을 설정할 수 있습니다.

CSS3에서는 border-radius, box-shadow, transform, transition, @keyframes, media query 같은 기능이 강화되었습니다.

7. 목록 스타일

list-style-type을 사용하면 목록의 불릿 모양을 바꿀 수 있습니다.

  • disc 기본 목록

  • none으로 불릿 제거 가능

8. display 속성
